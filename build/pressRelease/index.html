Babylon.js Welcomes Havok Physics

April 20, 2023

Babylon.js is excited to welcome a new physics implementation by Havok to the platform. The Babylon.js team in collaboration with the Havok team have been hard at work rewriting the existing physics interface in Babylon.js. This brings an improved, easier to use, and more flexible API to all developers/creators. The new interface ships with an implementation developed in collaboration with the Havok team raising the quality and performance bar for real time simulation in Babylon.js.

Babylon.js is on a mission to create one of the most powerful, beautiful and simple web rendering engines in the world. This sets a new benchmark for physics performance on the web, allowing developers and creators to unlock new functionality, create higher quality simulations, and dream up exciting new interactive experiences for users across the web.

The release of Babylon.js 6.0 with the first physics implementation by Havok brings:

  • Significantly improved performance
  • Expanded features for physics in Babylon.js
  • Improved physics debugging

This is a start of a partnership which will improve physics for all customers and future releases will look to grow the physics possibilities within the Babylon.js engine.

Babylon.js 6.0 is available from April 20th. To see the new physics implementation in action, check out the Babylon.js 6.0 Announcement Video and to get started visit us at https://www.babylonjs.com.

About Babylon.js

Babylon.js is a powerful, open-source, web rendering engine built to make it as simple as possible for web developers to unlock the power, performance, and rendering capabilities of the GPU onboard every device. Babylon.js works across all browsers and platforms and is completely free.

About Havok

Havok offers the fastest, most robust collision detection and physical simulation technology available, which is why it has become the gold standard within the games industry. With more than two decades of continuous investment and experience working with the top game studios in the world, Havok Physics has been validated by the best of the best and has powered hundreds of the top AAA titles on all major consoles.
